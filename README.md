# BOURBON MORELLI - Site E-commerce

Plateforme e-commerce moderne pour la vente de créations de couture haut de gamme à l'international.

## 📋 Description

BOURBON MORELLI est un site web dédié à la vente en ligne de vêtements et accessoires de couture premium. Le projet offre une expérience utilisateur fluide, sécurisée et professionnelle avec une gestion administrative complète.

## 🏗️ Architecture du Projet

```
bourbon-morelli/
├── client/                 # Frontend React
│   ├── public/
│   ├── src/
│   │   ├── components/     # Composants UI
│   │   ├── pages/         # Pages principales
│   │   ├── hooks/         # Hooks personnalisés
│   │   ├── services/      # Services API
│   │   ├── utils/         # Utilitaires
│   │   └── styles/        # Styles CSS/Tailwind
│   └── package.json
├── server/                # Backend Node.js/Express
│   ├── controllers/       # Contrôleurs API
│   ├── models/           # Modèles de données
│   ├── routes/           # Routes API
│   ├── middleware/       # Middleware
│   ├── config/           # Configuration
│   └── package.json
├── database/             # Scripts SQL
└── docs/                # Documentation
```

## 🚀 Technologies

### Frontend
- **React 18** avec hooks modernes
- **TailwindCSS** pour le design responsive
- **Lucide React** pour les icônes
- **React Router** pour la navigation
- **Axios** pour les requêtes API

### Backend
- **Node.js** avec **Express.js**
- **MySQL** comme base de données
- **JWT** pour l'authentification
- **Multer** pour l'upload d'images
- **Stripe & PayPal** pour les paiements

## 🔧 Installation

1. Cloner le projet
2. Installer les dépendances :
   ```bash
   npm run install-all
   ```
3. Configurer la base de données MySQL
4. Démarrer le développement :
   ```bash
   npm run dev
   ```
